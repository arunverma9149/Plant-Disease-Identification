# Plant Disease Identification

## About the Project

This project uses Deep Learning to identify plant diseases from leaf images. The model is trained on the PlantVillage dataset and can classify different plant diseases based on image features.

## Dataset

**Dataset:** PlantVillage Dataset

The dataset contains healthy and diseased leaf images from multiple crops with a total of **38 classes**.

## Technologies Used

- Python
- TensorFlow
- Google Colab
- NumPy
- Matplotlib
- GitHub

## Workflow

1. Load and preprocess the dataset.
2. Resize images to 224 × 224.
3. Split data into training and validation sets.
4. Train a CNN model.
5. Evaluate model performance.
6. Predict plant diseases from leaf images.

## Model Configuration

| Parameter | Value |
|------------|--------|
| Image Size | 224 × 224 |
| Batch Size | 32 |
| Epochs | 10 |
| Optimizer | Adam |

## Results

The model was successfully trained on the PlantVillage dataset and was able to classify plant diseases across 38 categories.
